INTERNET

Apa itu Internet?

Sebuah website itu seperti buku, dan halamannya disebut "halaman web". Seperti halnya buku bisa memiliki jumlah halaman yang berbeda, website juga bisa punya jumlah halaman web yang berbeda. Website bisa punya satu hingga ribuan halaman web. Setiap halaman web bisa berisi teks, gambar, video, atau konten lainnya yang berbeda.



Bagaimana cara kerja Internet?

Prosesnya cukup sederhana. Pertama, kamu memasukkan nama domain atau alamat website di kolom pencarian. Lalu, browser mengirim permintaan ke server DNS. Server DNS bertindak seperti direktori alamat, mengubah alamat yang bisa dibaca manusia menjadi alamat IP yang bisa dibaca mesin.

Setelah itu, permintaan dikirim ke server utama tempat situs disimpan. Server kemudian memberikan respons ke browser, dan website pun bisa diakses. Seluruh proses ini hanya memakan waktu 1-2 detik.

PROTOKOL

Apa itu HTTP?

Hypertext transfer protocol (HTTP) adalah sebuah protokol atau seperangkat aturan komunikasi untuk komunikasi klien-server. Saat Anda mengunjungi situs web, peramban Anda mengirimkan permintaan HTTP ke server web, yang merespons dengan respons HTTP. Server web dan peramban Anda akan bertukar data sebagai plaintext. Singkatnya, protokol HTTP adalah teknologi dasar yang mendukung komunikasi jaringan.



Apa itu HTTPS?

Seperti namanya, hypertext transfer protocol secure (HTTPS) adalah versi atau perpanjangan HTTP yang lebih aman. Di HTTPS, peramban dan server akan membuat koneksi yang aman dan terenkripsi sebelum mentransfer data.



Bagaimana cara kerjanya?

HTTP adalah protokol lapisan aplikasi yang mengatur komunikasi antara browser dan server melalui permintaan dan respons, seperti HTTP GET untuk mengambil data dan HTTP PUT untuk mengirim data. Responnya ditandai dengan kode, seperti 200 (OK) atau 404 (tidak ditemukan). HTTPS adalah versi aman dari HTTP, menggunakan SSL/TLS untuk mengenkripsi data, sehingga informasi yang dikirim tidak dapat dibaca pihak ketiga. Dalam HTTPS, browser memverifikasi sertifikat SSL dari server sebelum data terenkripsi dipertukarkan, memastikan komunikasi yang aman.

DOMAIN

Apa itu Domain Web?

Domain adalah alamat atau nama yang menjadi identitas website atau komputer agar bisa lebih mudah diakses oleh pengguna. Untuk website, nama domain biasanya diakhiri oleh ekstensi seperti .com atau .net.

Tanpa nama ini, Anda harus mengetikkan serangkaian angka yang disebut IP address di kolom alamat browser setiap kali akan mengunjungi sebuah website. Tentu saja ini akan sangat merepotkan, karena ada jutaan website di dunia ini dengan IP address yang berbeda-beda.



Bagaimana cara kerja Domain Web?

Cara kerja domain dimulai saat pengguna memasukkan nama domain di browser, yang kemudian menghubungi server DNS global. Server DNS mencari alamat IP yang sesuai dengan domain tersebut. Setelah ditemukan, server DNS mengirimkan informasi alamat IP ke browser. Selanjutnya, browser mengirim permintaan data ke server hosting domain. Server hosting merespons dengan data yang diminta, dan browser menampilkan halaman web kepada pengguna.

HOSTING

Apa itu Hosting Web?

Web hosting adalah tempat untuk menyimpan file dan data website agar bisa diakses lewat internet, sedangkan domain adalah alamat yang Anda gunakan untuk membuka website di browser.

Layanan web hosting memungkinkan Anda memublikasikan website ke Internet. Penyedia web hosting menyimpan data website Anda di servernya, lalu mengirimkan informasi tersebut ke web browser pengunjung saat mereka mengetikkan nama domain Anda ke address bar.



Bagaimana cara kerja Hosting Web?

Cara kerja hosting dimulai ketika sebuah website diunggah ke server penyedia hosting. Server ini bertindak sebagai tempat penyimpanan semua file dan data website. Ketika seseorang mengakses website melalui browser, permintaan tersebut dikirim ke server hosting. Server kemudian menanggapi dengan mengirimkan data yang diminta, seperti halaman web, gambar, atau video, ke browser pengguna. Hosting memastikan bahwa website dapat diakses secara online setiap saat, dengan menjaga performa server agar tetap optimal dan aman dari gangguan.

DNS

Apa itu DNS?

Sistem Nama Domain (DNS) adalah buku telepon Internet. Manusia mengakses informasi daring melalui nama domain , seperti nytimes.com atau espn.com. Peramban web berinteraksi melalui alamat Protokol Internet (IP) . DNS menerjemahkan nama domain ke alamat IP sehingga peramban dapat memuat sumber daya Internet.

Setiap perangkat yang terhubung ke Internet memiliki alamat IP unik yang digunakan oleh mesin lain untuk menemukan perangkat tersebut. Server DNS menghilangkan kebutuhan manusia untuk mengingat alamat IP seperti 192.168.1.1 (dalam IPv4), atau alamat IP alfanumerik baru yang lebih kompleks seperti 2400:cb00:2048:1::c629:d7a2 (dalam IPv6).



Bagaimana cara kerja DNS?

Proses resolusi DNS melibatkan konversi nama host (seperti www.example.com) menjadi alamat IP yang mudah diakses komputer (seperti 192.168.1.1). Alamat IP diberikan ke setiap perangkat di Internet, dan alamat tersebut diperlukan untuk menemukan perangkat Internet yang sesuai - seperti alamat jalan yang digunakan untuk menemukan rumah tertentu. Saat pengguna ingin memuat halaman web, penerjemahan harus dilakukan antara apa yang diketik pengguna ke peramban web mereka (example.com) dan alamat yang mudah diakses komputer yang diperlukan untuk menemukan halaman web example.com.

BROWSER

Apa itu Browser?

Browser adalah salah satu jenis perangkat lunak atau software yang pada umumnya di gunakan dan dimanfaatkan untuk membuka suatu halaman di situs internet. Browser juga di sebut dengan software yang berfungsi sebagai penerima, pengakses, dan penyaji berbagai informasi di internet.

Di dalam web browser terdapat konten berupa gambar, audio, video, dan halaman web. Konten-konten tersebut di hubungkan menggunakan hyperlink dan diklasifikasikan dengan URL (Uniform Resources Identifiers).



Bagaimana cara kerja Browser?

Ketika pengguna mengakses website dengan mengetikkan URL seperti youtube.com di address bar, web browser menerima permintaan tersebut dan mengambil data dari server DNS. Data yang diambil berupa alamat IP dari URL yang dimasukkan (www.youtube.com). Setelah web browser mendapatkan IP tersebut, browser mengakses server terkait. Server kemudian mengirimkan data konten, seperti HTML, PHP, dan CSS dari www.youtube.com. Akhirnya, web browser menampilkan konten yang sesuai dengan permintaan pengguna.
